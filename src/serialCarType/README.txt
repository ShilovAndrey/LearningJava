
Задание:

Создать класс CarType с переменными, описывающими марку, мощность, набор опций.
Сохранить 3 объекта в файл.
Из сохраненного файла вывести в консоль мощность первого объекта, марку второго и список опций третьего

Описание:

1. создается класс CarType, реализующий интерфейс Serializable.
 - класс содержит переменные: int power, String brand, String[] options.
 - класс переопределяет конструктор, чтобы при создании объекта класса необходимо было сразу указать все характеристики
 - класс реализует геттеры на каждую переменную.

2. в тестовом классе создаем функцию записи объектов в файл
 - создается низкоуровневый поток для связи с файлом FileOutputStream.
 - он соединяется с цепным исходящим потоком ObjectOutputStream, который сохраняет объекты в файл.
 
3. в тестовом классе создаем функцию извлечения объектов из файла
 - создается низкоуровневый поток для связи с файлом и считывания информации из него FileInputStream
 - создается цепной входной поток ObjectInputStream
 - создаются переменный CarType для записи в них информации
 - на экран выводится информация по каждому восстановленному экземпляру класса на выбор, через геттеры
