
Задача:

Создать программу, которая будет предлагать пользователю выполнить задачу, рандомно выбранную из уже изученного материала

Программа должна:

- узнавать у пользователя из какого направления он хотел бы получить задачу для решения
- после обратки ответа пользователя произвольно выбирать задачу из списка предоставленных файлов, считать выбранный файл
- выводить на экран задачу из файла
