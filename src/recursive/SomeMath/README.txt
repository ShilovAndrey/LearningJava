
Задание:

- создать класс с функцией, возвращающей значение факториала переданного числа
(используются два варианта исхода событий - 1 обычный и второй рекурсия)

- создать класс с функцией, выводящей на экран обратный отсчет от переданного в функцию значения
(1 путь - есть i<=1, второй путь - рекурсия с вычитанием в передаваемом значении)

- создать список. Вызвать функцию автоматичеки наполняющую этот список. Просуммировать все числа переданного списка и вернуть сумму
(создается функция sum с параметром типа списка и возвращаемым типом int. Если список пустой, то возвращается 0)

- найти наибольшее число в списке
