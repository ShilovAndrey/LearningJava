
Задание: 

1. записать в текстовый файл - "Какой цвет получится, если красный и синий?/фиолетовый".

2. извлечь из текстового файла, разделив на 2 строки, символом "/"

Решение:

1.
- создать file.txt, который будет являться адресом к месту нахождения файла
- создать поток передачи символов к файлу FileWriter
- создать буфер. записать в файл строку - "Какой цвет получится, если красный и синий?/фиолетовый".
2.
- создать FileReader
- подключить BufferedReader
- ссылке на объект класса String присвоить null
- в цикле while выводить на экран содержимое файла, пока строка не будет пуста
