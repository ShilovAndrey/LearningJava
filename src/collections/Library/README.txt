
Задание:

В библиотеке данные о выданных книгах записываются в файл "Список_книг.txt", где указывается название книги, автор, рейтинг(субъективная оценка). В течении дня книгу мшгут взять несколько раз, поэтому записи повторяются

Необходимо:
- обработать файл и внести все песни в список
- отсортировать список по алфавиту (Collections.sort)
- отсортировать список по рейтингу (Comparator)
- в множестве не должно быть дубликатов (HashSet)
- множество должно быть отсортировано по алфавиту (TreeSet)
- множество должно быть отсортировано по авторам (TreeSet + Comparator)
