
Задача:

- создать программу, которая произвольно выбирала бы текстовые файлы с заданиями для отработки навыков, считывала выбранный файл и выводила на экран задачу из файла

Описание:

- создать класс Coacher
- у класса есть переменная типа String[], в которой хранятся названия всех файлов с заданиями
- рандомно выбрать один из файлов из папки с заданиями, вывести название на экран
- подключиться к файлу с выбранным заданием и выводим инструкции на экран
